# Compile

## Compiler

### gcc

#### How to use
- `-l` option: library linking
  - without prefix `lib` and suffix `.a` or `.so`
  - `-llibrary`
- `-L` option: optional library directory
  - defualt library directory: `/lib`, `/usr/lib`, and `/usr/local/lib`
  - `-Ldirectory`
