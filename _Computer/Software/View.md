# View

### Combo Box
#### Basic
선택항목이 숨어있지 않고 화면에 처음부터 노출되어 있다. 사용자가 텍스트를 입력할 수 있다.

#### Dropdown
선택항목이 숨어있어 화살표 버튼을 누르면 펼쳐져서 사용자가 선택할 수 있다. 텍스트를 입력할 수 있다.

#### Dropdown list
선택항목이 숨어있다 콤보박스를 선택하면 선택항목이 펼쳐져 선택할 수 있다. 텍스트를 입력할 수 없다. 선택항목 중에 한가지만 선택 가능하다.
